⚠️ SE HOUVER PROBLEMAS COM NPM

Windows tem problemas com espaços no path. Se npm install falhar:

Solução Rápida:
   npm install --force --legacy-peer-deps

Solução Alternativa (use Yarn):
   npm install -g yarn
   yarn install
   yarn dev

Mais soluções:
   Abra SETUP_TROUBLESHOOTING.md
